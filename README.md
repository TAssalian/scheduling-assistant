# Usage
Place Google Gemini key in .env <br/>
cd my-app <br/>
npm install <br/>
npm run dev <br/>

# Overview
- Developed a React, AI agent web application that integrates Google Calendar with natural speech commands powered by Supabase OAuth, Google Gemini and Web Speech API, reducing event management time by 62%.<br/>

- Implemented a back-end pipeline to process user requests via Google Gemini API, outputting structured JSON to Google Calendar.
